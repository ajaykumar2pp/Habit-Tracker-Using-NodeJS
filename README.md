# 🌟 Habit Tracker

## ⭐ Introduction
This is a Habit Tracker project built using Node.js, EJS, CSS, JavaScript, and Bootstrap. It is responsive across all devices.

<br/>

## 🛠️ Features
- ➕ Add Habit<br/>
- 📅 Track Each Habit Weekly<br/>

<br/>

## 📦 Packages to Install
- `ejs`
- `express`
